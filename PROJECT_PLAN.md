# 📋 PROJECT_PLAN.md
# 미국-한국 주식 연계 자동매매 시스템 — 프로젝트 계획서

> **이 파일은 Claude Code와 협업용 살아있는 계획서입니다.**
> 진행 상황을 업데이트하며 관리하세요.
> 설계 변경은 CLAUDE.md → 이 파일 순서로 반영합니다.

---

## 🗓️ 전체 로드맵

```
Phase 1 (MVP)   : 1~4주차  → 모의투자 자동매매 작동 확인
Phase 2 (확장)  : 5~8주차  → 전체 11개 에이전트 완성
Phase 3 (검증)  : 9~12주차 → 실전 전환 전 4주 모의 안정화
Phase 4 (실전)  : 13주차~  → 소액 실전 시작 + 전략 누적
```
