OpenAPI를 활용한 투자전략 개발 편의성 강화를 위해,

당사 Open Trading API 공식 GitHub에 전략 생성·검증 기능 및 AI Extension이 신규 추가되었습니다.



이번 업데이트는 기존 단순 API 호출 예제를 넘어,

투자전략 생성과 백테스트 기능을 제공하고, AI 도구를 활용한 개발 지원까지 포함한 구조로 구성되었습니다.
